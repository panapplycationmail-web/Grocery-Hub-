# 🏪 Grocery Hub

एक संपूर्ण किराना शॉपिंग एप्लिकेशन जिसमें Shopping Cart और Checkout features हैं Web और Mobile दोनों प्लेटफॉर्म के लिए।

## 📱 Features

- ✅ Product Catalog
- ✅ Shopping Cart (Add/Remove/Update Quantity)
- ✅ Checkout Process
- ✅ Payment Integration (Stripe)
- ✅ Order Tracking
- ✅ User Authentication
- ✅ Delivery Address Management
- ✅ Order Confirmation

## 🏗️ Project Structure

```
Grocery-Hub/
├── backend/          # Node.js/Express API
├── web/              # React Web Application
├── mobile/           # React Native Mobile App
└── docs/             # Documentation
```

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Web Frontend**: React.js
- **Mobile**: React Native
- **Payment**: Stripe API
- **Authentication**: JWT
